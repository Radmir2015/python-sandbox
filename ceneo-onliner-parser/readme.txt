Инструкция по установке и запуску парсера:

1) Скачать Python 3: https://www.python.org/ftp/python/3.6.2/python-3.6.2.exe
	При установке выбрать добавление в $PATH или Command Prompt Line (чтобы проще запускать команды)

2) Распаковать все из архива в какую-нибудь папку

3) Открыть коммандную строку, зайти в папку, где лежит программа:
	Либо написать:
		cd /d <путь в папке>
	Либо в проводнике нажать правой клавишей мыши по папке и выбрать "Открыть окно команд"

4) Написать в ней:
	pip install -r requirements.txt
	или:
	python -m pip install -r requirements.txt
	// Это нужно, чтобы установились все модули использованные в программе

5) В currencies.xlsx можно поменять курс, либо поставить 1 в Update, чтобы произвести обновление при следующем запуске программы
	В main.xlsx внести нужную информацию, обязательно проверьте, чтобы названия файлов совпадали (в которых нужно обновлять цену),
	если будет отличие хотя бы в 1 символ, то обновления не будет

6) Для запуска программы вписать в консоль:
	python scrapsite.py
